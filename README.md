# Aurora Stream

Modern Streaming Theme für Jellyfin 10.11.x

## Installation

Ordnerstruktur

css/
    variables.css
    main.css
    navigation.css
    cards.css
    home.css
    animations.css
    responsive.css

Danach alle Dateien in eine einzige CSS-Datei zusammenführen oder über Custom CSS einbinden.

## Reihenfolge

variables.css

main.css

navigation.css

cards.css

home.css

animations.css

responsive.css

## Unterstützt

- Jellyfin 10.11.x
- Desktop
- Tablet
